# ECON 101 Educational Apps

Interactive web-based applications for teaching introductory economics concepts.

## Apps

- **crusoe-friday-trade.html** - Trade, MRS, and gains from exchange
- **PPF-bootstrap.html** - Production Possibility Frontier with factor allocation

## Getting Started

To build a new app, see **[BOOTSTRAP_APP_GUIDE.md](BOOTSTRAP_APP_GUIDE.md)** for:
- Standard template
- Critical patterns (solution pre-population, KaTeX usage)
- Best practices
- Common pitfalls

## Shared Resources

- **shared-bootstrap-edu.css** - Shared styles (Solarized Light theme)
- All apps use Bootstrap 5.3.0 + KaTeX for math rendering

## Quick Start

1. Copy the template from BOOTSTRAP_APP_GUIDE.md
2. Replace app-specific content
3. Follow the critical patterns (especially solution pre-population!)
4. Test with the checklist

Open any .html file directly in a browser - no server needed!
